# Optimized AVL

Optimized AVL tree, makes AVL trees for specific car make, and stores them in TreeMap.

### Usage
- Compile 
```bash
javac SaleRecord.java
javac myAVL.java
javac program8.java
```
- If you prefer you can use just program8
```bash
javac program8.java
```
- Run 
```bash
java program8 car_sales_data.csv test.csv
```
