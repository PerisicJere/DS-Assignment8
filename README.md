# Optimized AVL

Optimized AVL tree, makes AVL trees for specific car make, and stores them in TreeMap.

### Usage
- Compile from source file 
```bash
javac salerecordpack/*.java

```
- Run 
```bash
java salerecordpack.program8 car_sales_data.csv test.csv

```
